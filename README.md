# medical-chat-bot

# How to run ?
### steps 

clone the repo

``` bash
git clone https://github.com/shubh593/medical-chat-bot.git
```

## step 1 : create conda envoirnment after opening the repo

``` bash
conda create -n medibot python=3.10 -y
```

## step 2 : activate the conda envoirnment

``` bash
conda activate medibot
```

## step 3 : install the requirements

``` bash
pip install -r requirment.txt
```
